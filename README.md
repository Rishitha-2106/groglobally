# 🌐 GroGlobally - Official Website

This repository contains the official live website developed for **GroGlobally** during my **3-month internship**.  
The website is built using **pure HTML, CSS, and JavaScript**, focusing on a clean, responsive design and smooth user experience.

---



## 🛠️ Tech Stack
- **HTML5** – Structure and layout  
- **CSS3** – Styling and responsive design  
- **JavaScript (Vanilla JS)** – Dynamic interactivity and effects  

---

## ✨ Features
- Responsive layout optimized for mobile, tablet, and desktop  
- Modern UI with smooth animations and transitions  
- Sections for company overview, services, and contact details  
- Interactive navigation and form handling  
- Lightweight and fast-loading design
